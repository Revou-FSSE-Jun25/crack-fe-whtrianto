# Frontend .env example

## Development (Local)
```
VITE_API_URL=http://localhost:5000
```

## Production
```
VITE_API_URL=https://crackbe-production.up.railway.app
```

**Catatan:** 
- Untuk development lokal, buat file `.env` di folder `bookease-frontend` dengan nilai development
- Untuk production di Vercel, set Environment Variable `VITE_API_URL` di Vercel Dashboard dengan nilai production
